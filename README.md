# courier
Automated email notification client

```
courier | 信使 0.1
Enomothem
sent email.

USAGE:
    courier.exe [OPTIONS] --key <key> --report <output.html> --server <xxx.com> --to <xxx@xxx.com> --user <xxx@xxx.com>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -b, --bcc <xxx@xxx.com>       指定密送人（可选）
    -c, --cc <xxx@xxx.com>        指定抄送人（可选）
    -k, --key <key>               指定密码
    -r, --report <output.html>    指定报告路径
    -s, --server <xxx.com>        指定服务器
    -t, --to <xxx@xxx.com>        指定收件人
    -u, --user <xxx@xxx.com>      指定账号
```
